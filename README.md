## 简介

IC 现货 管理系统

## 安装使用

- 获取项目代码

```bash
git clone http://git.dzsc.com:3000/Weiku/ZXIC.Manage.git
```

- 安装依赖

```bash
cd ZXIC.Manage

yarn install
or
npm install
```

- 运行

```bash
yarn serve
or
npm run serve
```

- 打包

```bash
yarn build
or
npm run build
```
